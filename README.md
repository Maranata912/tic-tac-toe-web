# Tic Tac Toe Web Game

This is a **Tic-Tac-Toe game** you can play directly in the browser!  

## Features
- 2 players (X and O)  
- Restart button  
- Nice background color and hover effects  
- Shows winner or draw  

## How to play
1. Open `index.html` in your browser  
2. Click on the squares to place X or O  
3. Click "Restart Game" to play again
